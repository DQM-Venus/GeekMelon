# 新增数据源接入说明

当前已经新增并接入以下两个来源：

## 1. 量子位

- 运行代号：`qbitai`
- 默认入口：`https://www.qbitai.com/feed`
- 当前状态：已接入并加入 `china_mix`

单独运行：

```powershell
cd D:\project\GeekMelon\crawler
python main.py --source qbitai
```

可选配置：

```powershell
$env:GM_QBITAI_FEED_URL='https://www.qbitai.com/feed'
$env:GM_QBITAI_MAX_ITEMS='8'
```

## 2. 机器之心

- 运行代号：`jiqizhixin`
- 当前状态：代码已接入，但官网公开站点已切换到数据服务引导页，默认没有可直接抓取的公开列表
- 处理方式：保留为“可配置源”，当你拿到公开 feed 或私有数据服务入口后，设置环境变量即可启用

单独运行前请先配置：

```powershell
$env:GM_JIQIZHIXIN_FEED_URL='你的机器之心 RSS 或数据服务入口'
cd D:\project\GeekMelon\crawler
python main.py --source jiqizhixin
```

补充说明：

- `china_mix` 默认只会自动接入当前可稳定运行的 `量子位`
- 若后续配置了 `GM_JIQIZHIXIN_FEED_URL`，`china_mix` 也会自动带上机器之心
