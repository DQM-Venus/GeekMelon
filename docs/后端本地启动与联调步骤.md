# 后端本地启动与联调步骤

## 1. 建立 SSH 隧道

```powershell
ssh -L 3307:127.0.0.1:3306 root@47.105.112.190
```

保持该窗口不要关闭。

## 2. 启动后端

进入后端目录执行：

```powershell
cd D:\project\GeekMelon\backend
.\mvnw.cmd spring-boot:run
```

如果需要显式指定 Token，可先执行：

```powershell
$env:GM_INGEST_TOKEN='geekmelon-dev-token'
```

## 3. 发起联调请求

可直接使用：

- [ingest-demo.http](D:\project\GeekMelon\backend\http\ingest-demo.http)

或者使用 curl：

```powershell
curl.exe -X POST "http://127.0.0.1:8080/api/internal/feeds/ingest" ^
  -H "X-GeekMelon-Token: geekmelon-dev-token" ^
  -H "Content-Type: application/json" ^
  --data-binary "@D:\project\GeekMelon\backend\http\ingest-demo.json"
```

## 4. 预期响应

首次写入时：

```json
{
  "code": 0,
  "message": "接收成功",
  "data": {
    "id": 1,
    "action": "created"
  }
}
```

同一条数据重复发送时：

```json
{
  "code": 0,
  "message": "接收成功",
  "data": {
    "id": 1,
    "action": "skipped"
  }
}
```

若同一 `source + source_post_id` 的 `content_hash` 发生变化，则会返回：

```json
{
  "code": 0,
  "message": "接收成功",
  "data": {
    "id": 1,
    "action": "updated"
  }
}
```
