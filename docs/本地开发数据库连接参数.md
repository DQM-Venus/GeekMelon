# Geek Melon 本地开发数据库连接参数

## 1. 当前连接方式

当前本地开发环境通过 SSH 隧道连接云服务器上的 MySQL。

建立隧道命令：

```powershell
ssh -L 3307:127.0.0.1:3306 root@47.105.112.190
```

说明：

- 需要保持该 SSH 窗口持续开启
- 本地访问 `127.0.0.1:3307` 时，实际会转发到云服务器的 `127.0.0.1:3306`
